# Send to Ouaga

A modern web app for sending remittances to mobile money wallets in Ouagadougou, Burkina Faso (Orange Money, Mobicash, Telecel Money).

## Features
- User registration with email or phone
- KYC verification (placeholder)
- Connect payment methods (Stripe/Flutterwave for demo)
- Send money to BF mobile wallets
- Transaction history

## Tech Stack
- Frontend: Next.js 15 + Tailwind + TypeScript
- Backend: Next.js API routes
- Database: Prisma + PostgreSQL (setup later)
- Auth: NextAuth.js
- Payments: Flutterwave recommended for Africa

**Legal Note**: This is a starter template. Real money movement requires proper licensing, KYC/AML compliance, and partnerships with mobile money operators or aggregators like pawaPay, Tola Mobile.
